This M0 module scaffold declares only the approved Gradle convention plugins for an Android application + Compose + Hilt module. Runtime module dependencies are added in later capability PRs.

## Public APIs
No public API is implemented in LIVIO-M0-004.

## Testing
src/test/ and src/androidTest/ are tracked as empty source-set placeholders. Tests are added with the capability that introduces behavior.
